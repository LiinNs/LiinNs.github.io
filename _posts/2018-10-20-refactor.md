给项目添加sitemap
<!-- more -->
~~~ code
# 1. add this to Gemfile
group :jekyll_plugins do
  gem 'jekyll-sitemap'
end

# 2. add this to _config.yml file
plugins:
  - jekyll-sitemap

# 3. then go to workspace and hit
bundle
jekyll build
~~~
